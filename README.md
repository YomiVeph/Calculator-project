        Scientific Calculator
A responsive web-based scientific calculator built with HTML, CSS, and JavaScript. It supports basic arithmetic and advanced mathematical operations with a calculation history panel 

         Features
Scientific Functions (Sqrt, power, cos, sin, tan, log, pi, and factorial)
Basic operations: (+, -, *, /)
Calculation history
Keyboard Support
Allowing AC and CE to clear and delete entry
Responsive Layout

      Technologies Used
- HTML
- CSS
- JavaScript (DOM manipulations and eent handling)
